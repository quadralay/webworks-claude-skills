# ePublisher Claude Code Skills

Claude Code Skill for WebWorks ePublisher AutoMap integration - enabling AI-assisted documentation generation and format customization.

## Overview

This skill integrates WebWorks ePublisher AutoMap CLI with Claude Code, allowing users to:
- Run AutoMap commands through natural language requests
- Customize ePublisher project files (asp/scss/xsl)
- Navigate ePublisher's file resolver hierarchy
- Automate documentation generation workflows
